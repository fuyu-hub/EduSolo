# Guia do Usuário - EduSolos

Este documento fornece orientações básicas para a utilização das ferramentas integradas na plataforma **EduSolos**.

## 1. Introdução

O EduSolos foi desenvolvido para ser intuitivo. Cada módulo segue uma estrutura consistente:
- **Entrada de Dados:** Formulários para inserção de parâmetros laboratoriais.
- **Visualização em Tempo Real:** Gráficos e resultados calculados instantaneamente.
- **Exportação:** Opções para salvar curvas e relatórios.

---

## 2. Como Utilizar os Módulos

### 2.1 Índices Físicos e Limites
1. Insira os pesos e volumes solicitados.
2. Observe o **Diagrama de Fases** atualizar automaticamente.
3. Para os **Limites de Atterberg**, insira o número de golpes e a umidade correspondente. O gráfico de liquidez será traçado para determinar o LL.

### 2.2 Granulometria
1. No formulário de peneiramento, insira a massa retida em cada peneira da série selecionada.
2. Utilize o botão para **adicionar dados de sedimentação** caso o solo contenha finos.
3. A **Curva Granulométrica** será exibida à direita, juntamente com a classificação preliminar (porcentagem de pedregulho, areia, silte e argila).

### 2.3 Compactação
1. Insira os dados de massa de solo úmido e o teor de umidade para cada ponto do ensaio (geralmente 5 pontos).
2. Selecione a **Energia de Compactação** utilizada (Normal, Intermediária ou Modificada).
3. A **Curva de Compactação** destacará o ponto de umidade ótima e o peso específico seco máximo.

---

## 3. Funcionalidades Extras

- **Dados de Exemplo:** Clique no botão "Carregar Exemplo" para ver como os dados devem ser preenchidos e como os resultados são apresentados.
- **Exportação para PDF:** Utilize o ícone de impressora ou botão "Exportar" para gerar um relatório técnico formatado da sua análise atual.
- **Navegação:** Utilize a barra lateral para alternar entre os módulos sem perder o progresso atual.

---

## 4. Dicas Importantes

- **Validação de Campos:** Campos em vermelho indicam valores fora da faixa esperada ou campos obrigatórios vazios.
- **Precisão:** Certifique-se de utilizar a unidade de medida indicada em cada campo (ex: g, kg, cm³, kN/m³).
- **Relatórios:** Antes de exportar, verifique se todos os dados foram preenchidos corretamente para garantir a integridade do documento gerado.
